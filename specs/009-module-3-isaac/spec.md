## 1. Overview
This module focuses on **Advanced Perception and Training** using the NVIDIA Isaac platform. It bridges the gap between basic simulation (Gazebo) and photorealistic, AI-driven simulation (Isaac Sim). Students will learn to use Isaac Sim for synthetic data generation and Isaac ROS for hardware-accelerated perception.

### User Scenarios
- **Student** wants to understand why Isaac Sim is needed over Gazebo.
- **Student** needs to learn how to generate synthetic data for training computer vision models.
- **Student** wants to implement VSLAM and navigation using Isaac ROS.
