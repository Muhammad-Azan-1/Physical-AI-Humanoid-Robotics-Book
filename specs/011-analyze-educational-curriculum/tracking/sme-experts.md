# Subject Matter Experts for Physical AI Curriculum Validation

## Expert Categories

### ROS 2 Experts
- **Focus**: Robot Operating System 2 fundamentals, architecture, and best practices
- **Required Knowledge**: ROS 2 nodes, topics, services, actions, launch files, parameters
- **Potential Experts**: ROS 2 maintainers, experienced ROS 2 developers, robotics researchers

### Simulation Experts
- **Focus**: Gazebo, Isaac Sim, Unity simulation environments
- **Required Knowledge**: Physics simulation, sensor simulation, robot dynamics, sim-to-real transfer
- **Potential Experts**: Simulation software developers, robotics sim researchers

### AI/ML Experts
- **Focus**: Vision-Language-Action models, reinforcement learning for robotics
- **Required Knowledge**: Deep learning, computer vision, natural language processing, embodied AI
- **Potential Experts**: AI researchers, machine learning engineers, robotics AI specialists

### Humanoid Robotics Experts
- **Focus**: Bipedal locomotion, humanoid control systems, anthropomorphic design
- **Required Knowledge**: Humanoid kinematics, balance control, walking algorithms, safety systems
- **Potential Experts**: Humanoid robotics researchers, Boston Dynamics engineers, Unitree specialists

### Educational Experts
- **Focus**: STEM pedagogy, curriculum design, learning assessment
- **Required Knowledge**: Educational theory, learning outcomes, assessment methods
- **Potential Experts**: STEM educators, curriculum designers, educational researchers

## Validation Process
1. Identify specific content sections requiring expert validation
2. Match section to appropriate expert category
3. Request review using standardized review criteria
4. Document feedback and suggested changes
5. Implement validated improvements
